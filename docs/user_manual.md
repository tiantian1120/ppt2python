# 出国（境）学习交流项目展示系统 - 用户操作手册

## 1. 系统简介

本系统是"粤港澳大湾区人工智能与跨学科创新实践访学项目"的在线展示平台，由兰州城市学院重点扶持。系统全面展示了访学项目的背景信息、学习计划、参访安排、学生成果和个人档案等内容。

## 2. 功能导航

### 2.1 首页

首页是系统的核心入口，包含以下内容区域：

- **项目概览**：展示项目名称、副标题和项目徽章
- **统计数据**：以数字卡片形式展示国家级获奖数、省级获奖数、总成果数和专利软著数
- **精选成果**：展示最具代表性的竞赛获奖、专利和科研项目
- **学习计划预览**：展示四大核心学习模块的概要信息
- **参与动机**：展示参加访学项目的六大核心动机
- **预期成果**：展示访学后期望达成的学术与个人发展目标

### 2.2 项目介绍

通过顶部导航栏的"项目介绍"下拉菜单，可以访问以下子页面：

| 页面 | 说明 |
|------|------|
| 项目概览 | 查看所有项目列表，点击可查看项目详情 |
| 学习计划 | 查看四大核心学习模块的详细内容和课程安排 |
| 参访安排 | 查看粤港澳大湾区企业及高校参访计划 |
| 参与动机 | 了解参加访学项目的六大核心动机 |
| 预期成果 | 了解访学后期望达成的各项成果目标 |

### 2.3 成果展示

成果展示页面提供以下功能：

- **分类筛选**：按成果类型筛选（竞赛、专利、软著、项目、荣誉）
- **级别筛选**：按获奖级别筛选（国家级、省级、校级）
- **成果详情**：点击成果卡片可查看详细信息

### 2.4 个人档案

个人档案页面展示学生的完整信息：

- **基本信息**：姓名、专业、政治面貌、综合排名
- **统计数据**：各级别获奖数量统计
- **个人简介**：详细的学术背景和研究经历
- **技能标签**：掌握的技术技能列表
- **研究方向**：感兴趣的研究领域
- **获奖概要**：主要获奖情况总结
- **相关成果**：关联的竞赛和科研成果

### 2.5 申请指南

申请指南包含以下子页面：

| 页面 | 说明 |
|------|------|
| 申请指南首页 | 指南概览和快速导航 |
| 申请流程 | 七步申请流程的时间线展示 |
| 申请材料 | 完整的申请材料清单和说明 |
| 常见问题 | 8个常见问题的详细解答 |

### 2.6 联系我们

提供以下联系方式：

- 学校地址
- 电子邮箱
- 联系电话
- 办公时间
- 在线留言表单

## 3. 页面交互说明

### 3.1 导航栏

- **桌面端**：鼠标悬停在"项目介绍"上可展开下拉菜单
- **移动端**：点击右上角汉堡菜单图标展开导航，点击"项目介绍"可展开/收起子菜单

### 3.2 滚动动画

页面内容在滚动到可视区域时会自动显示入场动画，提供流畅的浏览体验。

### 3.3 成果筛选

在成果展示页面：
1. 点击顶部分类按钮（竞赛/专利/软著等）可按类型筛选
2. 点击级别按钮（国家级/省级/校级）可按级别筛选
3. 两个筛选条件可同时使用

### 3.4 常见问题手风琴

在常见问题页面：
1. 点击问题标题可展开/收起答案
2. 同时只能展开一个问题，点击其他问题会自动收起当前问题

### 3.5 响应式布局

系统支持桌面端、平板和手机三种屏幕尺寸，页面布局会自动适配。
